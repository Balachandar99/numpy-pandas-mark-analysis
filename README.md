# Student Mark Analysis using NumPy & Pandas

## Project Overview
This project analyzes marks of 100 students using NumPy and Pandas.

The project includes:

- Subject-wise Analysis
- Average Marks Calculation
- Highest & Lowest Scores
- Pass/Fail Analysis
- Student Performance Insights
